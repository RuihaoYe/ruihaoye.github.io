Type Assertion

type assertion信息编译后会消除。

// 支持两种形式的assertion
const v1 = v as number
const v2 = <number>v
// 经过两次assertion，转为任意类型
const a = (expr as any) an T

// 变量名后跟感叹号，表示non null check
return a! * 10

Narrowing

typeof

可能的返回值: string, number, bigint, boolean, symbol, undefined, object(包含array, null), function，没有null

if

返回false的值：0, NaN, “”, 0n(bigint 0), null, undefined。

可以适用Boolean()函数直接返回值的逻辑值。

==

== null或者== undefined，等价于=== null || === undeinfed

never

如果已经排除了所有可能，那么值类型会被判定为never。

// renamed import
import { pi as p } from './math.js'

// export default和export可以混合使用
export const pi = 3.15
export default function helloWorld() { ... }

import hello, { pi as p } from './math.js'
