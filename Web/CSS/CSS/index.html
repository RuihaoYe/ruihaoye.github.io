CSS Value

MDN

Absolute length units

如 cm、pt 以及 px

Relative length units

  • em: 在 font-size 中，指向上级的 font size；在 width 中，指向自身的 font size
  • rem: Font size of the root element.
  • vw, vh, vmin, vmax: viewport 的 1%，或 1%的 viewport 小值、大值

function

.box {
  width: calc(20% + 100px);
}

Overflow

overflow: visible(默认值);
overflow: auto(自动判断是否展示滚动条) hidden(隐藏内容) scroll(强制加上滚动条);
// 单独设定x方向、y方向
overflow-x / overflow-y: auto;

size

  • intrinsic size
  • extrinsic size

在 margin、padding 中使用百分比时，百分比乘的是父 container 的 inline size（也就是父元素不包含 padding、border、margin 的 width）。

box model

inline box 的几个注意点：

  1. 垂直方向的 padding、margin、border 会生效，但不会把别的元素挤远，水平方向的照常；
  2. width 和 height 不会生效

inline-block：不会另起一行，同时会使上面 inline box 的两点失效。

Layout

display: block, inline, inline-block, flex, grid,

flex

main axis和cross axis的概念。

// flex container
.container {
  display: flex;
  flex-direction: [row column row-reverse column-reverse];
  flex-wrap: [<null> wrap]; // wrap要与flex搭配使用
  flex-flow: <flex-direction> <flex-wrap>;
  align-items: [strech center flex-start flex-end]; // cross axis
  justify-content: [flex-start flex-end center space-around space-between]; // main axis
}

.item {
  flex: <flex-grow> <flex-shrink> <flex-basis>; // 每个元素在main axis上至少flex-basis宽
  flex-grow: 1; // 默认值为1
  flex-shrink: 1; // 默认值为1。当所有item的宽之和超过container的宽度，items则会按shrink值缩小
  align-self: [strech center flex-start flex-end]; // override container's align-items
  order: 1; // 默认值为0，相同order的元素按原始顺序排，其他按order值大小排，可为负

  flex: initial; // 等于 "flex: 0 1 auto"
  flex: auto; // 等于 "flex: 1 1 auto"
  flex: none; // 等于 "flex: 0 0 auto"
}

position

  • static: default
  • relative: relative to its position in normal flow, 其他元素的排列中依然给这个元素留下了空位
  • absolute: relative to container, 其他元素的排列不给这个元素留下空位
  • fixed: fix an element relative to browser viewport
  • sticky
